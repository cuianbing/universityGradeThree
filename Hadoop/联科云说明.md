
7. 使用sqoop导入数据
```shell?linenums
cd /opt/sqoop/bin/
sqoop import --connect jdbc:mysql://192.168.0.200:3306/retail  --username root 
--password 123456 --table 表名 --hive-import --create-hive-table -m 1

```
